# 🤖 Agentic AI Finance & Personal Assistants

A collection of **Agentic AI systems** built using **Agno**, designed to solve real-world problems in **finance** and **personal productivity**.

This project demonstrates how autonomous AI agents can **analyze, decide, and act** using tools like web search, financial APIs, and custom logic.

---

## 🚀 Features

### 💰 Finance Agents

* 📊 Stock market analysis
* 📈 Real-time financial insights
* 💡 Investment suggestions (data-driven)
* 📰 News-based market sentiment analysis

### 🧠 Personal AI Agents

* 📅 Task & productivity assistant
* 🔍 Smart web search agent
* 🧾 Information summarizer
* 🤖 Multi-tool reasoning agent

---

## 🛠️ Tech Stack

* **Agno** (Agent framework)
* **Groq API** (LLM inference)
* **Python**
* **DuckDuckGo Search Tools**
* **YFinance API**
